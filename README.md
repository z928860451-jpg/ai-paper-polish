# 笔润 BiRun · AI论文降重降AI味平台

学术文本润色服务，把AI生成的论文段落改写为自然、人类化、低查重率的学术表达。

## 技术栈

- **前端/后端**：Next.js 14（App Router，全栈一体）
- **数据库**：SQLite + Prisma（小流量起步够用，后期可换Postgres）
- **大模型**：DeepSeek API（deepseek-chat模型，按token计费，零硬件投入）
- **支付**：虎皮椒（个人开发者最快接通微信+支付宝）

## 核心策略（已内置）

降AIGC策略写在 `lib/anti-aigc-prompt.ts`：

- 禁用30+AI高频词与套话
- 句长杂揉、主动语态、自然过渡
- 句式重组、同义改写、数据表达变形
- 保留学术术语与引文，仅改表达

## 启动

```bash
# 1. 安装依赖
npm install

# 2. 复制环境变量并填写
cp .env.example .env
# 编辑 .env，填入 DeepSeek API Key、JWT密钥、虎皮椒商户信息

# 3. 初始化数据库
npx prisma db push

# 4. 启动开发服务器
npm run dev
```

打开 http://localhost:3000

## 计费规则

- 单价：0.05元/字（去空格后字符数）
- 新用户：500字免费试用
- 套餐：¥9.9 / ¥45 / ¥128 / ¥398 四档（可在 `lib/billing.ts` 调整）

## 部署建议

- **零硬件投入**：DeepSeek API按token计费，1000字成本约0.015元，售价50元，毛利率99%
- Web服务器：阿里云2核4G足够初期，约80元/月
- 备案：国内域名要ICP备案，要不然支付通道不给你接
- 合规：网站底部要标明"仅供学术润色辅助，不替代原创"
