### 核心降AIGC策略（已内置）
写在 `lib/anti-aigc-prompt.ts`，作为system prompt注入每次模型调用：

**硬性禁令**
- 30+AI高频词黑名单（深入探讨/扮演重要角色/delve into等）
- 禁破折号、禁"不是...而是..."、禁三段排比
- 禁单句段落、禁连续"此外/而且"开头

**改写原则**
- 句长杂揉（短句8-15字 + 长句30-45字混排）
- 主动语态优先
- 用具体数据替代笼统词
- 保留学者主观判断口吻

**降查重技巧**
- 句式重组、同义改写、结构变换
- 数据表达变形（阿拉伯数字↔中文、百分比↔区间）
- 专业术语保留语义不动

### 收费链路完整闭环
1. 用户注册 → 自动赠500字免费额度
2. 粘贴文本 → 实时显示字数与费用
3. 提交润色 → 检查余额（不够弹充值）→ 调本地大模型 → 写回数据库 → 扣费
4. 充值 → 选套餐 → 调虎皮椒下单 → 跳支付页 → 异步回调入账
5. 长文自动分段（单段≤2500字）防止上下文溢出

## 下一步行动清单

### 上线前必做
1. **启动本地大模型**：找一台4090或A100机器跑vLLM
   ```bash
   python -m vllm.entrypoints.openai.api_server \
     --model Qwen/Qwen2.5-14B-Instruct --port 8000
   ```
2. **注册虎皮椒商户**：https://www.xunhupay.com，拿APPID/APPSECRET
3. **域名+ICP备案**：国内支付必须备案
4. **填写`.env`**：复制`.env.example`改名为`.env`，填全
5. **初始化数据库**：`npx prisma db push`
6. **本地跑通**：`npm run dev` → 访问 http://localhost:3000

## 商业模式速算
- 单字成本：本地模型几乎为零，电费摊到约0.001元/字
- 单字售价：0.05元/字
- 毛利率：~98%
- 一个4090月租2000元，能撑住每月约200万字处理量
- 客单价9.9-398元，转化100单/月即可覆盖服务器成本
